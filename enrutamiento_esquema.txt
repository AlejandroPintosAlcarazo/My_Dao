Esquema de Rutas Web

Landing Page

    Ruta: /
    Descripción: Página principal de la web.


About Us

    Ruta: /about-us
    Descripción: Información sobre la DAO y su misión.


Guilds

    Ruta: /guilds
    Descripción: Página principal de las guilds (gremios).


Guild Specific

    Ruta: /guilds/:guildId
    Descripción: Detalle de una guild específica.
    Ejemplo: /guilds/ciencias, /guilds/artes, /guilds/idiomas


Missions

    Ruta: /guilds/:guildId/missions
    Descripción: Listado de misiones para una guild específica.
    Ejemplo: /guilds/ciencias/missions, /guilds/artes/missions


Mission Specific

    Ruta: /guilds/:guildId/missions/:missionId
    Descripción: Detalle de una misión específica dentro de una guild.
    Ejemplo: /guilds/ciencias/missions/1, /guilds/artes/missions/2


Profile

    Ruta: /profile
    Descripción: Perfil del usuario.
    Notas: Podría tener subrutas como /profile/edit para editar el perfil.


Proposals

    Ruta: /proposals
    Descripción: Página de propuestas de gobernanza DAO.
    Ejemplo: /proposals/create para crear una nueva propuesta.


Proposal Specific

    Ruta: /proposals/:proposalId
    Descripción: Detalle de una propuesta específica.
    Ejemplo: /proposals/1, /proposals/2


Chat

    Ruta: /chat
    Descripción: Página de discusiones y chat entre usuarios.



Consideraciones Adicionales

Autenticación y Autorización: Asegúrate de que las rutas que requieren autenticación (por ejemplo, /profile, /proposals, /chat) estén protegidas y solo accesibles para usuarios registrados.

Panel de Administración: Si necesitas un panel de administración para gestionar las guilds, misiones, y propuestas, considera añadir rutas como /admin/guilds, /admin/missions, etc.

Errores y Redirecciones: Implementa rutas para manejar errores 404 (no encontrado) y redirecciones necesarias.
